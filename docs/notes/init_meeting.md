# Project Requirements: Pharmacy Management

Business is a pharmacy with the following problems.

Application User:

1. Admin
2. Client (patients)
3. Staff Members of Business

## Billing and Inventory Management

* Create a way to select medicines from a database with inventory levels.

## Reading Prescription

* Use OCR from Gemini API
* Dosage and Medicine name with description

## Prescription Medicine

* Use LLMs to understand the symptoms of patients and prescribe medicine.
* Maybe have a chat-like interface.

## Medicine Requests

* Embed spreadsheet for specific medicine requests.

## Data Dashboard

* Display a tab for basic analytics.

## Reminder for Re-Order

* Re-stocking reminders for staff members.

## Book Doctor's Appointment

* Add-on feature for patients to book an appointment for registered doctor.
* Appointments show-up in admin or staff account.
